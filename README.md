# VisMan
An entry management web application

## Demo
Go to a live demo @ [VisMan](https://vis-man.herokuapp.com/)
## Note
 - Ensure port 465 is allowed through your firewall for email through Gmail (certainly, if on public networks). Hence, prefer mobile hostspots or home connections.
 
 - SMS feature will not work on all numbers due to limitations on Nexmo trial account. Number should be OTP verified under Nexmo account for the SMS to deliver.
